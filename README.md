# Case Taxi Tip Prediction
> IBM GPXX0AXIEN
> 
> Case criado e guiado pela IBM

**Link do dataset:** https://cf-courses-data.s3.us.cloud-object-storage.appdomain.cloud/IBM-GPXX0AXIEN/data/yellow_tripdata_2019-06_1M.tgz

#### **Descrição do Case:**
Diga que está dirigindo um táxi, Uber ou Lift e quer descobrir quanto de gorjeta receberia por uma corrida. Neste projeto, você criará um modelo de IA para fazer exatamente isso. Utilizará Scikit-learn para construir modelos de Regressão.

**Modelo utilizado por mim:** HistGradientBoostingRegressor

- MAE:  0.7026
- MSE:  1.5533
- RMSE: 1.2463
- R²:   0.7825
